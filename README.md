## 👨‍💻 Tentang Saya

Saya **AI Engineer** dan mahasiswa Sains Data (Semester 4) di **UIN Salatiga**, dengan fokus pada irisan antara **Artificial Intelligence** dan **Software Engineering**. Bukan sekadar melatih model — saya membangun sistem *end-to-end*: dari arsitektur model, *vector database*, hingga *backend API* yang siap dipakai di produksi dan diakses *real-time* oleh aplikasi web.

Lulusan **Coding Camp Foundation (Dicoding)**, aktif mendalami **LLM architecture**, **Retrieval-Augmented Generation (RAG)**, dan **Deep Learning** — dengan target karier sebagai **AI/ML Engineer** atau **AI Backend Engineer** di perusahaan teknologi.

> 💡 **Value proposition saya:** Saya menjembatani gap yang sering terjadi antara *data scientist* yang jago model dan *engineer* yang jago sistem — saya bisa keduanya, dari notebook sampai ke production API.
