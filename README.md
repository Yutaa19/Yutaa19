Ketertarikan utama saya terletak pada titik temu antara Kecerdasan Buatan dan *Software Engineering*—saya tidak hanya melatih model AI, tetapi juga membangun infrastruktur dan sistem *backend* agar model tersebut dapat diakses dan digunakan secara *real-time* dalam aplikasi web.
